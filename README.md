
This repository contains a Task Management application built using Java and Spring Boot.
The project is designed as a comprehensive internship project to demonstrate proficiency
in building RESTful APIs, using best practices in Java development, and creating scalable backend solutions.

## Features 

- Add new tasks with title, description, and due date
- Edit existing tasks
- Delete tasks
- Mark tasks as completed or pending
- View tasks by status (pending/completed)
- RESTful API design using Spring Boot
- Data persistence (using JPA/Hibernate with an SQL database)
- Exception & error handling
- Simple authentication (if implemented)
- Organized code structure for maintainability

## Technologies Used

- **Language:** Java
- **Framework:** Spring Boot
- **ORM:** Spring Data JPA / Hibernate
- **Database:** MySQL 
- **Build Tool:** Maven 
- **Other:** Lombok for boilerplate reduction 

### Prerequisites

- Java 8 or above
- Maven
- Spring Boot
-  MySQL server
